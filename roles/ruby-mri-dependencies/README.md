* [zlib1g](http://packages.ubuntu.com/search?keywords=zlib1g)
* [zlib1g-dev](http://packages.ubuntu.com/search?keywords=zlib1g-dev)
* [libssl-dev](http://packages.ubuntu.com/search?keywords=libssl-dev)

* [libxml2-dev](http://packages.ubuntu.com/search?keywords=libxml2-dev)
